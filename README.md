
# 💰 Finova

Finova is a modern fintech web application that helps users manage investments, track portfolios, and gain AI-powered financial insights through a clean and responsive interface.

## ✨ Features

- User Authentication
- Secure Dashboard
- Portfolio Tracking
- AI Financial Insights
- Dark Mode
- Responsive Design

## 🛠 Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Supabase
- OpenAI API

## 🚀 Getting Started

```bash
npm install
npm run dev
```

Open `http://localhost:3000` in your browser.
